An example of how you can animate the background based on scroll motion.
This card is incredibly long, just so we can see the scrolling in action.
(As a bonus we also have this neat little glass effect. It's stupidly simple to do, but it looks pretty cool.)
The syntax for that is simply this:
backdrop-filter: blur(10px);
This creates this glass thing backgrop we see. We can of course also use some other filter, and in fact, the MDN has a full page on filters.
https://developer.mozilla.org/en-US/docs/Web/CSS/filter
